# Teacher Dashboard - Quick Reference Card

## Quick Access
1. Open Worms Math Game
2. Click **"LÆRER LOGIN"** (bottom-right corner)
3. Enter teacher username
4. Click **"LOG IND"**

## Dashboard Controls

| Button | Function |
|--------|----------|
| **OPDATER** | Refresh data from server |
| **LOG UD** | Logout and return to lobby |
| **Navn** | Sort students alphabetically |
| **Spørgsmål** | Sort by total questions attempted |
| **Korrekt %** | Sort by accuracy percentage |
| **Vis kun elever <70% korrekt** | Show only struggling students |

## Color Coding

| Color | Meaning | Action |
|-------|---------|--------|
| 🟢 **Green** | >80% correct | Excellent - student is doing well |
| 🟡 **Yellow** | 60-80% correct | Good - monitor progress |
| 🔴 **Red** | <60% correct | Needs help - intervention recommended |
| 🔴 **Red Background** | Weakest topic | Focus extra support here |

## Reading the Table

### Format: `X/Y (Z%)`
- **X** = Correct answers
- **Y** = Total attempts
- **Z%** = Accuracy percentage

**Example:** `15/18 (83%)` means:
- 15 questions answered correctly
- 18 questions attempted total
- 83% accuracy for this topic

## Topics Tracked

1. **Division** - Divisionsspørgsmål
2. **Brøker** - Brøkregning (fractions)
3. **Geometri** - Geometriske beregninger
4. **Problemregning** - Tekstopgaver (word problems)

## Common Tasks

### Find Struggling Students
1. Click **"Vis kun elever <70% korrekt"**
2. Review red-highlighted students
3. Check their weakest topics (red background)

### Identify Class Weaknesses
1. Look at the bottom summary: "Gennemsnitlig nøjagtighed"
2. Sort by **"Korrekt %"** (lowest first)
3. Note common weak topics across multiple students

### Track Progress
1. Note current accuracy percentages
2. Click **"OPDATER"** periodically
3. Compare new percentages with previous
4. Look for improvements or declines

### Generate Report (Manual)
1. Sort students as needed
2. Take screenshot (Print Screen key)
3. Paste into document/email
4. Or manually copy data to Excel

## Interpreting Results

### Student Performance Levels

| Accuracy | Assessment | Recommendation |
|----------|------------|----------------|
| 90-100% | Mastery | Challenge with harder problems |
| 80-89% | Proficient | Continue practice, minor support |
| 70-79% | Developing | Targeted practice needed |
| 60-69% | Below Target | Intervention recommended |
| <60% | Significant Gap | Immediate support required |

### Topic-Specific Interventions

**Division (Low Scores):**
- Review division basics
- Practice times tables
- Use visual aids (arrays, groups)

**Brøker (Low Scores):**
- Use fraction circles/bars
- Practice equivalent fractions
- Real-world examples (pizza slices)

**Geometri (Low Scores):**
- Hands-on measuring activities
- Draw and label shapes
- Use manipulatives

**Problemregning (Low Scores):**
- Teach reading strategies
- Break problems into steps
- Practice identifying keywords

## Troubleshooting

| Problem | Solution |
|---------|----------|
| Can't login | Ensure you have a teacher account (contact admin) |
| "Kun lærere kan tilgå..." | You're using a student account - use teacher account |
| Empty dashboard | No students have played yet - they need to answer quiz questions |
| Usernames show "Ukendt" | Server connection issue - click OPDATER to retry |
| Old data showing | Click OPDATER to refresh |

## Tips & Best Practices

### Daily Routine
1. Login at start of class
2. Note overall class accuracy
3. Identify struggling students
4. Plan interventions

### Weekly Review
1. Compare current vs. previous week
2. Track improvement trends
3. Adjust teaching strategies
4. Celebrate student progress

### Intervention Planning
1. Sort by accuracy (lowest first)
2. Note students <70%
3. Identify their weakest topics
4. Group students by topic weakness
5. Plan small-group instruction

### Parent Communication
1. Take screenshot of student's row
2. Highlight specific weak topics
3. Share with parents
4. Provide practice suggestions

## Data Refresh Schedule

- **Real-time:** Dashboard shows data at time of login
- **Manual Refresh:** Click OPDATER anytime
- **After Quiz Match:** Students' data updates immediately after match ends
- **Recommendation:** Refresh every 30-60 minutes during class

## Session Management

- Dashboard stays logged in until you click LOG UD
- Closing browser logs you out
- Can access from any computer (web-based)
- Multiple teachers can login simultaneously

## Privacy & Security

- Only teachers with accounts can access
- Students cannot see other students' data
- Data is read-only (cannot edit)
- No student personal information shown (usernames only)
